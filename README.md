# Yatin Vanjani - Developer Portfolio

A cyberpunk-themed developer portfolio built with HTML, CSS, and JavaScript. Clean design with smooth animations and modern aesthetics.

## Features

- **Cyberpunk Design**: Neon colors with futuristic styling
- **Animated Background**: Dynamic grid and scanline effects  
- **Interactive Elements**: Hover effects and smooth transitions
- **Responsive Layout**: Works on all screen sizes
- **Professional Sections**: Skills, projects, services, and contact info

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript
- Google Fonts (Orbitron, Share Tech Mono)

## Quick Start

1. Download the files
2. Open `index.html` in any browser
3. Customize the content in the HTML and CSS files

## Customization

**Update your info:**
- Edit name and title in `index.html` (lines 15-17)
- Change avatar image (line 25)
- Update contact info (lines 420-430)

**Change colors:**
Edit the CSS variables in `styles.css`:
```css
:root {
    --neon-blue: #00f3ff;
    --neon-pink: #ff00ff;
    --bg-color: #050505;
}
```
